StudyTrack is a lightweight local study timer & dashboard for Linux (Omarchy). Runs locally and shows a dark responsive UI at `http://localhost:8080`.

## What you get (Phase 1)
- Start/stop server with a simple CLI
- Live timer + session naming + tags
- Sessions saved to local SQLite DB
- Dark responsive UI (Tailwind CDN)
- Runs detached in background so you can close the terminal

## Requirements
- Python 3.8+
- pip

Recommended: use a Python virtual environment.
